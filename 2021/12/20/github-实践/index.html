git 分支

master分支（现为nain分支）是Git默认创建的分支，因此基本上所有开发都是以这个分支为中心进行的。

只要创建多个分支，就可以在不互相影响的情况下同时进行多个功能的开发

git 远程库

远程仓库顾名思义，是与我们本地仓库相对独立的另一个仓库，让我们先在GitHub上创建一个仓库，并将其设置为本地仓库的远程仓库。
