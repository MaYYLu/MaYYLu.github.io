前言

为了建立数据结构和算法的一套标准，并且降低他们之间的耦合关系，以提升各自的独立性、弹性、交互操作性(相互合作性,interoperability),诞生了STL
标准模板库（Standard Template Library，STL），是一个C++软件库，大量影响了C++标准程序库但并非是其的一部分。其中包含4个组件，分别为算法、容器、函数、迭代器。

string

1．字符串定义函数
string str1默认初始化

str3 = str1复制 str1 到 str3

str3 = str1 + str2连接 str1 和 str2

string s4(n,'c')s4由n个c组成

swap(str1,str2) 替换字符串

reverse(str.begin(),str.end())翻转字符串

str3 = str1 + 'c'连接 str1 和 c

str[i]访问字符串中的字符（不会抛出异常）

str.at(i)访问字符串中的字符（不会抛出异常）

2．字符串操作函数
.size() || .length()返回str3 的总长度

.insert(插入的位置,插入的字符串) 插入字符

.erase(起始下标,删除子字符串的长度) 删除字符

.find(待查找的子字符串,开始查找的位置)字符串查找

.substr(起始下标,提取的子字符串的长度) 剪切字符串

.replace(起始下标,终止下标,替换的字符串)替换字符

vector(向量)

字符串操作函数

1.push_back 在数组的最后添加一个数据

2.pop_back 去掉数组的最后一个数据

4.begin 得到数组头的指针

5.end 得到数组的最后一个单元+1的指针

6.front 得到数组头的引用

7.back 得到数组的最后一个单元的引用

8.max_size 得到vector最大可以是多大

9.capacity 当前vector分配的大小

10.size 当前使用数据的大小

11.resize 改变当前使用数据的大小，如果它比当前使用的大，者填充默认值

12.reserve 改变当前vecotr所分配空间的大小

13.erase 删除指针指向的数据项

14.clear 清空当前的vector

15.rbegin 将vector反转后的开始指针返回(其实就是原来的end-1)

16.rend 将vector反转构的结束指针返回(其实就是原来的begin-1)

17.empty 判断vector是否为空

18.swap 与另一个vector交换数据
