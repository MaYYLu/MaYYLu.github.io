T3协议

Java RMI 的基础通信协议是 JRMP,而WebLogic T3协议是Oracle WebLogic Server中用于通信的协议之一。它基于Java语言，通过TCP/IP协议进行通信，通常使用在Java EE应用程序与WebLogic Server之间的通信中。

默认情况下，管理控制台的端口是 7001，应用是 7003，节点管理器是 5556，初次部署时需要注意让防火墙放行这三个端口。
